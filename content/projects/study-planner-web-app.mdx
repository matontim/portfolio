## Overview
A full-stack study planning application currently in active development. 
This project is being built to deepen practical experience with full-stack 
architecture, REST API design, database modeling, and deploying production 
web applications. The goal is a tool that helps users organize study 
sessions, track progress, and manage tasks across subjects.

> **Status:** In development: Task management 
> features are complete. User authentication, dashboard and analytics views are in progress.

[View the GitHub Repository](https://github.com/matontim/studydesk)

---

## Tech Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| Frontend | Next.js 15, TypeScript | UI and routing |
| Styling | Tailwind CSS | Component styling |
| Backend | Next.js API Routes | REST endpoints |
| Database | PostgreSQL | Persistent storage |
| ORM | Prisma | Database modeling |
| Auth | NextAuth.js | User authentication |
| Deployment | Vercel | Hosting and CI/CD |

---

## Architecture

The application follows a monorepo structure with the Next.js frontend 
and API routes colocated. The database layer uses Prisma ORM for 
type-safe queries and schema migrations.

---

## Current Features

  via NextAuth.js with JWT tokens
- **Task management**: create, edit, delete, and complete study tasks
- **Subject organization**: group tasks by subject with color coding
- **Responsive layout**: mobile-first design with Tailwind CSS
